# Chapter 3 Challenge: Branded Weather Widget

A custom JavaFX weather widget designed for "Aero Dynamics", featuring aviation-themed metrics and styling.

## Screenshot

![Widget Screenshot](widget_screenshot.png)

## Features
- Aviation-themed UI
- Custom aircraft vector graphic
- Flight condition metrics (Wind, Visibility)
- 3-Day Forecast

## How to Run
1. Ensure you have Maven installed.
2. Run the following command:
   ```bash
   mvn clean javafx:run
   ```
